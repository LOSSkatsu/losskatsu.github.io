# 1. 깃헙으로 작업하는 과정

* branch 따기
* pull 하면 로컬 코드가 변한다
<br />

* 작업...작업... 
<br />

* 스테이지에 올리기
* 커밋(commit)
* 푸쉬(push)
    * push 한 후 알파서버에 적용해본다. 이상 유무 체크
    * 이상이 없다면 다음 단계로

* 깃헙 홈페이지로 이동(내가 push한 branch로 이동) 
* pull request
* merge
* 작업 완료 한 branch 삭제
