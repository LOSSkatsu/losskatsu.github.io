# 동적 메모리 할당(malloc, free)

* 힙(heap) : 프로그래머가 관리하는 메모리 공간.

malloc는 인자로 전달된 정수 값에 해당하는 바이트 크기의 메모리 공간을 힙 영역에 할당하고, 
이 메모리 공간의 주소값을 반환함(할당된 메모리 공간의 첫번째 바이트를 가리킴).
