# 포인터(pointer) 이해하기

> 포인터 변수: 메모리의 주소 값을 저장하기 위한 변수

변수에는 여러가지 종류가 있죠. 
포인터 변수도 변수의 한 종류입니다. 
우리가 int 형 변수를 선언 한다고 하면

```c
int a;
int b = 5;
```

라고 입력하면 int형 변수 a가 선언되고, 
int형 변수 b에 5라는 값을 넣으면서 초기화도 가능합니다.
