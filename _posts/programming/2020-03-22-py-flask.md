## 가상환경 실행 및 flask 프레임워크 실행확인

저는 가상환경 이름을 py3_7_2라고 지었습니다.

```bash
$ pyenv activate py3_7_2
(py3_7_2)$ python
>>> from flask import flask
>>> app = Flask("test")
>>> app
<Flask 'test'> 
```
