# 커버로스(kerberos) 개념 정리

## 서버 접근 권한 관리 프로토콜

서버와 클라이언트 수가 아주 많아지면 서버마다 유저별 접근권한 관리하는게 힘들어진다. 
그래서 등장한 것이 커버로스(kerberos). 
커버로스를 이용하면 티켓을 가진 유저만 서버에 접속할 수 있도록 제어할 수 있으므로 서버 접근 권한 관리가 한결 편해진다.
