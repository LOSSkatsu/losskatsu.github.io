# 하둡(hadoop) 기본개념

빅데이터 시스템에서 가장 중요한 부분은 데이터 수집. 
수집 용도로 많이 쓰이는 오픈소스는 Flume, Chukwa 등. 
메세징 기반으로 구성된 Kafka를 사용하기도 함. 

* 하둡의 구성
    : HDFS(Hadoop Distributed File System)이라는 분산 파일 시스템
    : MapReduce 라고 부르는 분산 처리 시스템
